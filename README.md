# 语文老师小助手

> **核心声明：本工具唯一用户为小学 1-6 年级语文教师，无任何学生、家长使用入口与相关功能，全程仅服务教师个人教学教研工作。**

## 项目概述

本工具聚焦解决语文教师备课效率低、批改负担重、出题组卷难、教学资源管理乱四大核心痛点，围绕教师「备 - 教 - 批 - 研」全工作流，打造贴合 2022 版新课标、零合规风险的纯教学属性专属数字化辅助工具，为一线教师减负增效。

## 核心功能

### 1. 正版教材与备课中心
- 覆盖部编人教版、苏教版等主流版本 1-6 年级全学段正版教材
- 配套新课标同步教案、课件、范读音频等资源
- 支持个性化备课标注、教研组协同备课

### 2. 全场景教学辅助
- **课文多粒度拆分朗读**：支持分段、分句朗读
- **AI 朗读测评**：智能评测朗读流畅度、准确度
- **智能出题组卷**：题库管理、同类题生成
- **作文/客观题批改辅助**：AI 多维度评价
- **班级学情分析**：成绩统计、知识点掌握分析

### 3. 专属 AI 与资源管理
- **AI 教研助手**：备课、教学、批改、教研全场景智能辅助
- **个人教学资源库**：教案、课件、试题统一管理
- **专业成长档案**：培训、教研、荣誉记录

## 合规红线

- ✅ 所有教材、教学资源均获正版授权，无侵权风险
- ✅ 全程无广告、无弹窗、无违规功能
- ✅ 不收集未成年人信息
- ✅ 严守双减政策与数据安全法规
- ✅ 无学生/家长入口，仅限教师使用

## 技术栈

- **框架**: Next.js 14 + React 18
- **语言**: TypeScript
- **样式**: Tailwind CSS
- **状态管理**: Zustand
- **图标**: Lucide React
